## What is it
ColorCatcher is a simple Color Picker like the one in macOS, the difference is that this one shows the Hex and RGB values (unlike the macOS one that only does RGB colors)

When you have the perfect color and don't want to remember the code, you can just pin it to the top by pressing the pin button on the corner.

## Why
This app was made because I was trying to capture a color and it wasn't working well with the default macOS color picker, so I decided to make my own. No this app is NOT vibe coded.
